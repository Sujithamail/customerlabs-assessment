# CustomerLabs Assessment

## Objective

This project demonstrates an end-to-end modern data engineering workflow using Google BigQuery, SQL, dbt, Git, GitHub, and dashboarding tools.

The project includes:

- Data exploration using BigQuery
- SQL analysis
- Data transformation
- Data modeling using dbt
- Dashboard creation
- Documentation
- Git version control
